# Specification

## Summary
**Goal:** Replace all occurrences of the current brand name with "Helping Hand" across the entire application (frontend and backend), while preserving all other content and design.

**Planned changes:**
- Update Header logo/brand text to "Helping Hand"
- Update Footer brand name to "Helping Hand"
- Update HTML page title in index.html to "Helping Hand"
- Update About page, Terms & Conditions page, and Privacy Policy page to reference "Helping Hand"
- Replace any remaining brand name references in all other frontend components and pages
- Update backend Motoko actor to replace any hardcoded store/brand name strings with "Helping Hand"

**User-visible outcome:** The entire site consistently displays "Helping Hand" as the brand name everywhere it appears, with no trace of the old brand name remaining.
